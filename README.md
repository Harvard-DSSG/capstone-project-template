# Digital Humanities Capstone Project Template

This repository is a starter template for Digital Humanities Certificate capstone projects.

## Quickstart

- [Install NodeJS](https://nodejs.org/) if you haven't already
- `npm install`
- `npm run dev`

Eleventy serves the site locally and rebuilds on changes.

## Key folders

- `src/`: website content, layouts, and static assets
- `src/project/`: example pages you can delete or modify
- `src/notebooks/`: Jupyter notebooks (`.ipynb`) rendered to HTML during build
- `documentation/`: project documentation templates

## Customize

- Edit site title/description: `src/_data/site.json`
- Edit navigation: `src/_data/navigation.json`
